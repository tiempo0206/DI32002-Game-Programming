# April 24 Class Activity

This folder contains the Unity 3D interactive solar system project completed for the April 24 class activity.

The project includes:

- a 3D solar system scene with the Sun, eight planets, the Moon, rings, orbit lines, and an asteroid belt
- interactive selection and camera focus for celestial bodies
- child-friendly information panels and simple audio/visual feedback
- clearer script organization under `Assets/Scripts/SolarSystem/` for easier development and reading

Unity-generated folders such as `Library`, `Temp`, `Logs`, and `UserSettings` are intentionally excluded from version control.
